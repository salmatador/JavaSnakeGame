
This is the snake Game ReImagined By Micah Kline

It is a work in progress.

You are the snake you job is to eat the apples and grow

Watch out for yourself or you will die

this is only level one so you can go through walls as there are not any as of yet


F2 - starts a new game
F3 - Pauses / Resumes the game

Use Arrow Keys to move

Have fun

High score will reset when program is closed.....

****************************************************
Extract the zip file to a folder and open a command prompt

compile App.ProgramEntry.java

---- javac App.ProgramEntry.java

run the program

---- java App.ProgramEntry
